# 📊 Credit Risk Analysis & Default Prediction

## 🔍 Project Overview

This project focuses on analyzing customer financial data to identify **credit risk and loan default patterns**.
Using Python for data analysis and Power BI for visualization, the project helps in understanding which customers are likely to default and supports better credit decision-making.

---

## 🎯 Objectives

* Analyze customer financial behavior
* Identify high-risk customers
* Understand factors affecting loan default
* Build an interactive dashboard for decision-making

---

## 🛠️ Tools & Technologies

* Python (Pandas, Matplotlib, Seaborn)
* Power BI
* Excel / CSV Dataset

---

## 📂 Dataset

* Credit Risk Dataset (Kaggle)
* Contains customer details like income, loan amount, credit history, and default status

---

## ⚙️ Project Workflow

### 1. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized data formats

### 2. Feature Engineering

* Created **Risk Category** (Low, Medium, High)
* Calculated **Debt-to-Income Ratio (DTI)**
* Converted loan status into default flag

### 3. Exploratory Data Analysis (EDA)

* Default vs Non-default distribution
* Income vs Loan relationship
* Loan grade vs default rate
* Interest rate impact on default

### 4. Data Visualization (Power BI)

* Built interactive dashboard with:

  * KPI Cards (Total Customers, Default Rate, Avg Income)
  * Risk segmentation charts
  * Scatter plot (Income vs Loan)
  * Loan grade vs default analysis

---

## 📊 Key Insights

* Customers with **higher loan-to-income ratio** have higher default risk
* Lower income groups tend to default more
* Higher interest rates are associated with increased defaults
* Previous default history is a strong indicator of future risk
